# ComfyUI 工作流文档

本目录包含 ComfyUI 工作流的详细说明文档。

## 工作流列表

| 工作流 | 文件 | 说明 |
|--------|------|------|
| [Z-Image-Turbo](z-image-turbo.md) | `image_z_image_turbo.json` | 高效文生图，支持中英文 |
| [Hunyuan3D v2.1](hunyuan3d-v2.1.md) | `3d_hunyuan3d-v2.1.json` | 图片转 3D 模型 |

## 文档结构

每个工作流文档包含：

1. **概述** - 工作流功能简介
2. **结构图** - 节点连接关系可视化
3. **节点详解** - 每个节点的参数说明
4. **所需模型** - 需要下载的模型文件
5. **显存需求** - 硬件要求
6. **常见问题** - 问题排查

## 工作流文件位置

工作流 JSON 文件保存在: `Z:\user\default\workflows\`
