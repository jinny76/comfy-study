# 贡献指南

感谢你对这个项目的兴趣！欢迎任何形式的贡献。

## 如何贡献

### 报告问题

- 文档错误（错别字、代码错误、过时内容）
- 功能建议（希望添加的模块解析）
- 使用问题

请通过 [Issue](https://github.com/jinny76/comfy-study/issues) 提交。

### 提交 PR

1. Fork 本仓库
2. 创建分支 `git checkout -b feature/xxx`
3. 提交修改 `git commit -m 'Add xxx'`
4. 推送分支 `git push origin feature/xxx`
5. 创建 Pull Request

### 文档规范

- 使用中文编写
- Markdown 格式
- 代码块标注语言类型
- 源码引用注明文件路径和行号

### 目录结构

```
├── study/          # 源码解析笔记
├── docs/           # 用户指南
├── workflow-docs/  # 工作流文档
└── tools/          # 实用工具
```

## 联系方式

如有问题，欢迎通过 Issue 交流。
